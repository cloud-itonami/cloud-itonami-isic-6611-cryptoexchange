# Contributing

This is an Open Business Blueprint repo (maturity: **blueprint** — no actor
implementation yet). Contributions that advance it toward :implemented are
welcome: blueprint refinements, incident-register additions (a documented
exchange failure class not covered by INV-1..14 is a design bug — please
report it), jurisdiction spec-basis research, safety-kernel battery cases,
and the langgraph actor itself (kernels first, per ADR-2607121200
discipline). Open an issue or PR. License: AGPL-3.0-or-later.
